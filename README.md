# Karo Pitch - Landing Page

A responsive landing page built for **Karo Pitch**, a startup pitch platform by KaroStartup.

---

## Live Demo

[karo-pitch-g52m318hv-1sumans-projects.vercel.app](https://karo-pitch-g52m318hv-1sumans-projects.vercel.app)

---

## Tech Stack

- **React** 
- **Vite** 
- **Tailwind CSS v4**
- **Google Fonts**

## Getting Started

```bash
# Clone the repo
git clone https://github.com/1Suman/karo-pitch.git

# Move into the project
cd karo-pitch

# Install dependencies
npm install

# Start the dev server
npm run dev
```
